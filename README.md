# I Might Suck - X-Rated AI Guru

A brutally honest, profanity-laden AI Guru website that tells you if you suck at life.

## Setup

1. Get an API key from [Uncensored AI API](https://uncensored.com/api)
2. Open `script.js` and replace `YOUR_API_KEY_HERE` with your actual API key
3. Open `index.html` in your browser

## Features

- **Uncensored AI Responses**: Uses Uncensored AI API for dynamic, profanity-laden responses
- **Fallback System**: If API fails or no key is set, uses predefined responses
- **Brutal Honesty**: The Guru doesn't hold back
- **X-Rated Content**: Explicit language throughout

## Note

The API integration requires a valid API key. If you don't have one, the site will use the fallback response system with predefined answers.
