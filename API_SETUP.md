# How to Get an Unhinged API Key

Here are your best options for getting an uncensored/unhinged LLM API:

## Option 1: ApiFreeLLM.com (Easiest - No API Key Needed!)
- **Website**: https://www.apifreellm.com
- **Pros**: 
  - No registration or API key needed
  - Completely free
  - Unlimited requests (rate limit: 1 request per 5 seconds)
- **Cons**: Rate limited
- **How to use**: I'll update the code to use this - no setup needed!

## Option 2: ModelsLab Uncensored Chat API
- **Website**: https://docs.modelslab.com/uncensored-chat-api/uncensored-chat
- **How to get key**:
  1. Go to https://modelslab.com
  2. Sign up for an account
  3. Navigate to API section
  4. Generate your API key
- **Pros**: More reliable, higher rate limits
- **Cons**: Requires signup, may have costs

## Option 3: Together AI (Good for Uncensored Models)
- **Website**: https://together.ai
- **How to get key**:
  1. Sign up at https://together.ai
  2. Go to API Keys section
  3. Create a new API key
  4. Use models like `meta-llama/Llama-3-70b-chat-hf` or other uncensored models
- **Pros**: Reliable, good free tier, many uncensored models
- **Cons**: Requires signup

## Option 4: OpenRouter.ai
- **Website**: https://openrouter.ai
- **How to get key**:
  1. Sign up at https://openrouter.ai
  2. Go to Keys section
  3. Create an API key
  4. Use uncensored models like `meta-llama/llama-3-70b-instruct`
- **Pros**: Access to many models, good free tier
- **Cons**: Requires signup

## Option 5: Uncensored AI (Private Beta)
- **Website**: https://uncensored.com/api
- **How to get key**:
  1. Visit their API page
  2. Request access to private beta
  3. Wait for approval
- **Pros**: Specifically designed for uncensored content
- **Cons**: Private beta, may take time to get access

## Recommended: Start with ApiFreeLLM.com
Since it requires no setup, I'll update the code to use this by default. If you want better performance later, switch to Together AI or OpenRouter.

## After Getting Your API Key:
1. Open `script.js`
2. Find the line: `const API_KEY = 'YOUR_API_KEY_HERE';`
3. Replace `YOUR_API_KEY_HERE` with your actual API key
4. Update `API_URL` if using a different service
